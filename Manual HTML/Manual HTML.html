-INTRODUCCION

HTML, sigla en inglés de HyperText Markup Language (lenguaje de marcas de hipertexto), hace referencia al lenguaje de marcado para la elaboración de páginas web.
-EMPEZANDO

Para escribir HTML ncesitaremos de un procesador de texto que nos permita introducir las llamadas "etiquetas".Estas se definen entre corchetes angulares( < y >).Posteriormente , una vez escrito el documento lo identificaremos con
cualquier nombre que deseemos junto la extension .html
-COMPOSICION DE UN TEXTO HTML

Un texto HTML esta compuesto de etiquetas, los atributos de las etiquetas y la informacion que estas contienen.Estas etiquetas se abren constan de un elemento de apertura"<etiqueta>" y uno de cierre "</etiqueta>".Dentro de la etiqueta de
apertura se introduciran los atributos de las etiquetas,variables en cada una de ellas.

Ej: "<body background=red>"

Para comenzar una texto HTML , primero definiremos , que tipo de documento es con "<!doctype>", lo comenzaremos con "<html>", introduciremos una cabezera con "<head>" e insertaremos un cuerpo con "<body>". El resultado debió de quedar asi:

"<!doctype>"
<html>
<head>
</head>
<body>
</body>
</html>

A partir de aqui ya podras trabajar en el documento con las etiquetas y atributos que te presente a continuacion.
METADATOS (HEAD)

Dentro de etsa seccion entran los elementos que describen el contenido informativo de la etiqueta en la que estan indexados. La mayoria de ellos , por no decir su totalidad se encuentra en la etiqueta <head>.

<title> : Define un titulo para el documento.

<link>: es uno de los metadatos que pueden ir indexados a otras etiquetas aparte de la cabezera.Ofrece al autor la posibilidad de enlazar sus documentos a otros recursos.

<meta>: define un conjunto de valores que no pueden ser definidos por las anteriores etiquetas ,como el tipo de codificacion.

<style>: esta etiqueta hace referencia a la seccion de css , un lenguaje de formateado de nuestro documento que no explicaremos en este manual.
CUERPO DEL DOCUMENTO (BODY)

Esta seccion mostrara al usuario que abra la pagina la informacion que hemos escrito en ella, y la mostrara de una manera u otra segun las etiquetas utilizadas en ella.Las mas usadas son:

<article>

<seccion>

Ambas etiquetas son utilizadas para estructurar el contenido del documento.

<h>: Se utiliza para introducir una cabecera a diferentes partes del documento.

<header>: Se encargan de indicar a los navegadores que lo que contienen es la cabecera de la página.

<footer>: Se utiliza para indicar a los navegadores que lo que contienen es el pie de la página.

<p> : se utiliza para introducir parrafos.

<pre>: indica al navegador que se encuetra ante un texto preformateado, lo que significa que respeta una fuente de letra de ancho fijo , los espacios y los saltos de linea.

<li>: esta etiqueta se utiliza tanto en listas ordenadas(<ol>) como en listas desordenadas (<ul>).

LISTA ORDENADA:
  1. 1
  2. 2
  3. 3
LISTA DESORDENADA
  • 1
  • 2
  • 3


<hr>: representa una linea horizontal para separar el texto en diferentes partes.

<br>:Representa un salto de linea.

<br>:permite dividir palabras, haciendo un salto de línea si la palabra no entra en el navegador, evitando así el scroll horizontal.

<a>: Anchor: permite definir un destino o un origen de un enlace(Ejemplo).

<img>: permite definir una imagen para mostrar.

<table>: permite crear una tabla.

----<tr> :permite crear una fila.

----<th> :permite crear un encabezamiento.

----<td> :permite crear una celda.

ELEMENTOS DE RESALTADO(BODY) Son elementos utilizados para poner enfasis de diferente maneras en diferentes partes del texto modificando la aparencia de ciertos caracteres.

<i>: Bastardilla : Todos los caracteres en el interior de este elemento se veran en el navegador como cursiva.

<b>: Negrita : Todos los caracteres en el interior de este elemento se veran en el navegador como negrita.

<i>: Subrayada : Todos los caracteres en el interior de este elemento se veran en el navegador como cursiva.

<em>: Enfasis : Se usa para indicar enfasis, los navegadores suelen indicarlo con cursiva .

<strong>: Enfasis Fuerte : Se usa para indicar enfasis fuerte, los navegadores suelen indicarlo con negrita.

<small>: Letra Pequeña : Se usa para representar un texto con letra pequeña(como la de un contrato).

<s>: Contenido Obsoleto/Irrelevante : Se usa para representar un texto que esta desaprovado.(como la de un contrato).

ATRIBUTOS DE HEAD

-profile: Especifica la URL de un perfil de metadatos.

-dir: Especifica la dirección del texto.Valores podibles: ltr(de izquierda a derecha) y rtl(de derecha a izquierda).

-lang: Especifica el idioma base de los valores de los atributos y del texto contenido en un elemento.Ayuda a los motores de búsqueda.

ATRIBUTOS DE BODY

-class: Asigna un nombre de clase. Servira posteriormente para el manual de CSS

-style: Este atributo especifica información de estilo para el elemento actual.

-title: Este atributo ofrece información consultiva sobre el elemento para el cual se establece.

-dir: Especifica la dirección del texto.Valores podibles: ltr(de izquierda a derecha) y rtl(de derecha a izquierda).

-lang: Especifica el idioma base de los valores de los atributos y del texto contenido en un elemento.Ayuda a los motores de búsqueda.

ATRIBUTOS DE IMG

-width: Permite al navegador predeterminar el ancho, en pixles, que tendrá tu imagen.

-height: Junto con el atributo width, el navegador puede preparar el espacio necesario para tu imagen antes de que se muestre.

-border=x: Para añadir o eliminar un borde, donde la x será un valor numérico.

-align: Puedes alinear una imagen horizontal y/o verticalmente en una página usando este atributo.

-alt="descripción_de_la_imagen":El atributo alt te permite describir la imagen para los navegadores de sólo texto, así como etiquetar la imagen antes de que se cargue en una página.

ATRIBUTOS DE TABLE

-border=x: Para añadir o eliminar un borde, donde la x será un valor numérico.

-rowspan: Permite unir una celda con las celdas contiguas en vertical.

-colspan: Permite unir una celda con las celdas contiguas en horizontal.

ATRIBUTOS DE LI(lista)

-reversed: invierte el orden de los elementos de la lista,sea ordenada o desordenada.

-start: Define el elemento de la lista por el que esta empieza,,sea ordenada o desordenada.

-type: Define el indexado de la lista(numeros, letras mayusculas y minusculas...).
